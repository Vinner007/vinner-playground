# TIL 01 — GitHub Achievements

- GitHub profile achievements (Quickdraw, YOLO, Pull Shark, ...) only count activity in **public** repositories.
- Quickdraw: close an issue or PR within 5 minutes of opening it.
- YOLO: merge a pull request without a review.
- Pull Shark: have 2 pull requests merged (next tiers at 16 / 128 / 1024).
- Badges can take a few minutes up to about an hour to appear on your profile.
